# Sistema de Gestión de Inventario

Sistema completo para la gestión de inventario de productos con API REST, interfaz web y pruebas automatizadas.

## Descripción

Este proyecto implementa un sistema de inventario que permite gestionar productos y categorías. Incluye:

- API REST con arquitectura por capas (Modelo - Servicio - Controlador)
- Interfaz web desarrollada en Next.js
- Base de datos SQLite
- Pruebas unitarias, de integración y E2E
- Análisis estático de código
- Pipeline de integración continua

## Requisitos Previos

- Node.js 18 o superior
- npm
